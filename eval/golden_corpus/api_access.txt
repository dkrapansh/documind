API Access and Rate Limits

Northwind provides a REST API for programmatic file upload, download, and metadata access. API requests require a bearer token generated from the account settings page. The Standard plan is limited to 1,000 API requests per day; the Business plan allows 10,000 requests per day; the Enterprise plan has no fixed rate limit and instead uses a fair-use policy. Exceeding the daily limit returns an HTTP 429 response until the limit resets at midnight UTC.
